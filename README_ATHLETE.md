# Garmin Analytics — для атлета

Шорт-инструкция для атлета, которого тренер подключил к общей БД.

## Что ты получишь от тренера
1. **ZIP-архив проекта** или git-доступ.
2. **Твой ATHLETE_ID** — короткое имя на латинице (например, `ivan_petrov`).
3. **Креды Turso** — `TURSO_DATABASE_URL` и `TURSO_AUTH_TOKEN` (общие для всех атлетов тренера).

## Что нужно сделать (один раз)

### 1. Распаковать в удобное место
Например, `C:\garmin_analytics\`.

### 2. Установить Python 3.13
[python.org/downloads](https://www.python.org/downloads/release/python-3133/) — Windows installer 64-bit.

### 3. Установить зависимости
В папке проекта открой PowerShell или cmd:
```cmd
py -3.13 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### 4. Заполнить `.env`
Скопируй `.env.example` → `.env`, открой в Блокноте, заполни:
```
GARMIN_EMAIL=твой_email_от_garmin
GARMIN_PASSWORD=твой_пароль_от_garmin

ATHLETE_ID=ivan_petrov          # как сказал тренер
IS_ADMIN=false

TURSO_DATABASE_URL=libsql://garmin-xxx.turso.io   # от тренера
TURSO_AUTH_TOKEN=eyJhbGc...                       # от тренера
```
**Пароль остаётся ТОЛЬКО на твоём ноутбуке.** Тренер его не видит.

### 5. Первая синхронизация
Двойной клик по `sync.cmd`. При первом запуске Garmin может попросить **код двухфакторки** — введи его в консоли и Enter. Дальше пойдёт выгрузка тренировок за год (5–15 минут).

### 6. Проверить дашборд
Двойной клик по `dashboard.cmd` → откроется в браузере. Увидишь свои графики.

## Дальше — автоматически

После первой настройки тренер настроит у тебя автозапуск:
- **Sync ежедневно в 4 утра** — догружает свежие тренировки в облако
- **Дашборд при логине** — всегда доступен на `http://localhost:8501`

## Что хранится в облаке
Только **обработанные данные тренировок**:
- список тренировок (длительность, темп, пульс, HR-зоны)
- ежедневные метрики (шаги, RHR, стресс, body battery)
- сон (фазы, длительность, качество)
- HRV

**Никаких паролей, GPS-точек домашнего адреса, личной переписки.** Тренер видит только спортивные показатели.

## Если что-то не работает
- **«MFA / код подтверждения»** — введи в консоли, Enter
- **«database is locked»** — закрой дашборд, перезапусти sync
- **«Network error» при sync** — проверь интернет, повтори
- **Любые другие ошибки** — пришли тренеру скрин из консоли

## Удалить интеграцию
Если хочешь прекратить:
1. Скажи тренеру — он удалит твой `ATHLETE_ID` из БД.
2. Удали папку проекта.
3. В `Task Scheduler` удали задачи `GarminDashboard` и `GarminSync` (если были созданы).
