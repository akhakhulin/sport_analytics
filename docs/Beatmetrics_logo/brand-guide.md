# BeatMetrics — Brand Guidelines

Краткий гайд по использованию знака. Длинные правила не нужны — у нас простой бренд.

## Знак

Строчная **b** с пиком пульса справа на квадратной плашке с закруглёнными углами.

Метафора: «b» — Beat, начало бренда. Пик пульса — то, что приложение измеряет. Закруглённый квадрат — современная иконка приложения.

## Цвета

| Назначение | HEX | Где использовать |
|---|---|---|
| **Brand Purple** | `#3c3489` | Основной фирменный цвет. Плашка иконки, кнопки CTA, акценты. |
| **Light Purple** | `#7f77dd` | Тёмная тема. Гиперссылки. Hover-состояния на тёмном фоне. |
| **Ink** | `#1a1a18` | Основной текст. Знак на светлом фоне без плашки. |
| **Paper** | `#fdfdfc` | Фон лендинга и интерфейса. |
| **Mute** | `#6b6a64` | Вторичный текст, подписи. |
| **Line** | `#e8e7e1` | Разделители, бордеры. |

## Шрифт

Системный sans-serif. Конкретно: `-apple-system, "Segoe UI", Inter, system-ui, sans-serif`.

Заголовки и логотип — **Medium (500)**. Основной текст — **Regular (400)**.

Letter-spacing для логотипа: `-0.02em` (чуть плотнее обычного для лучшего «бренд-feel»).

В написании **BeatMetrics** обе части одного начертания — без выделения «Beat» или «Metrics» жирным. Это слитное название.

## Правила использования знака

**Можно:**
- Использовать на фирменном фиолетовом фоне
- Использовать на чёрном или белом фоне (монохромная версия)
- Изменять размер пропорционально
- Использовать как фавикон, иконку приложения, в шапке сайта, в подписях email

**Нельзя:**
- Менять пропорции (растягивать, сжимать)
- Менять цвет плашки на не-фирменный
- Добавлять обводки, тени, градиенты, блики
- Поворачивать знак
- Размещать на фотографиях без подложки (если нужно — добавьте полупрозрачный фон)
- Использовать знак отдельно от названия, если бренд ещё не узнаваем (первые 1–2 года) — всегда рядом текст

## Отступы (clear space)

Вокруг знака должна быть пустая зона, равная **половине высоты знака** со всех сторон. Никакие другие элементы (текст, кнопки, другие изображения) не должны заходить в эту зону.

## Минимальный размер

- Иконка приложения: 60×60 px
- Знак рядом с текстом: 24×24 px
- Только знак (без названия): 16×16 px (favicon)

Меньше — теряется читаемость пика пульса.

## Файлы

В папке `/svg/` — векторные исходники (для разработки и дизайнеров):
- `icon-primary.svg` — основной знак на фиолетовой плашке
- `icon-transparent.svg` — знак без плашки (фиолетовый на прозрачном)
- `icon-mono-black.svg` — чёрно-белая версия для печати
- `icon-dark-theme.svg` — версия для тёмной темы
- `horizontal-light.svg` — знак + текст BeatMetrics для шапки сайта (светлая тема)
- `horizontal-dark.svg` — то же для тёмной темы
- `favicon.svg` — оптимизированный для маленьких размеров

В папке `/png/` — растровые версии (для магазинов приложений и публикаций):
- `icon-1024.png` — для App Store (требует именно этот размер)
- `icon-512.png` — для Google Play
- `icon-180.png` — для iOS Home Screen
- `icon-192.png`, `icon-256.png` — для Android и веба
- `favicon-16.png`, `favicon-32.png`, `favicon-64.png` — фавиконы
- `icon-dark-theme-1024.png` — версия для tinted/dark App Store
- `mark-mono-black-1024.png` — монохромная для печати
- `mark-transparent-1024.png` — знак без плашки

## Использование в HTML

Фавикон в `<head>`:

```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/icon-180.png">
<link rel="manifest" href="/site.webmanifest">
```

`site.webmanifest` (для PWA и Android):

```json
{
  "name": "BeatMetrics",
  "short_name": "BeatMetrics",
  "icons": [
    { "src": "/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ],
  "theme_color": "#3c3489",
  "background_color": "#fdfdfc",
  "display": "standalone"
}
```

## Использование в CSS

```css
:root {
  --color-brand: #3c3489;
  --color-brand-light: #7f77dd;
  --color-ink: #1a1a18;
  --color-paper: #fdfdfc;
  --color-mute: #6b6a64;
  --color-line: #e8e7e1;
}

@media (prefers-color-scheme: dark) {
  :root {
    --color-brand: #7f77dd;
    --color-ink: #f7f6f1;
    --color-paper: #1a1a18;
    --color-mute: #96958e;
    --color-line: #2e2e2b;
  }
}
```

## Что дальше

Этот гайд — стартовая точка. Когда наймёте дизайнера, он может:
- Превратить знак в анимированную версию для splash-экрана (пик «бежит» сбоку буквы)
- Сделать пиктограммы в едином стиле для интерфейса
- Расширить палитру для дашбордов и графиков ЧСС-зон (5 цветов для зон 1-5)
- Сделать иллюстрации для лендинга в стилистике знака
