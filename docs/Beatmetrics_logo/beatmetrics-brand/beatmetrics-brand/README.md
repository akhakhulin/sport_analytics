# BeatMetrics — Brand Assets

Финальный пакет файлов логотипа.

## Что внутри

```
beatmetrics-brand/
├── svg/                          # векторные исходники
│   ├── icon-primary.svg          # основная иконка приложения (плашка + знак)
│   ├── icon-transparent.svg      # знак без плашки
│   ├── icon-mono-black.svg       # чёрно-белая для печати
│   ├── icon-dark-theme.svg       # для тёмной темы
│   ├── horizontal-light.svg      # горизонтальная для шапки сайта (light)
│   ├── horizontal-dark.svg       # горизонтальная для шапки сайта (dark)
│   └── favicon.svg               # оптимизирован для фавикона
│
├── png/                          # растровые для магазинов и веба
│   ├── icon-1024.png             # App Store (обязательный размер)
│   ├── icon-512.png              # Google Play
│   ├── icon-256.png              # macOS
│   ├── icon-192.png              # Android, веб
│   ├── icon-180.png              # iOS Home Screen
│   ├── icon-128.png, 96, 64      # резервные размеры
│   ├── favicon-16/32/64.png      # фавиконы
│   ├── icon-dark-theme-1024.png  # для тёмной темы магазинов
│   ├── mark-mono-black-1024.png  # монохром
│   └── mark-transparent-*.png    # знак без плашки
│
└── docs/
    └── brand-guide.md            # правила использования
```

## Быстрый старт

**Для сайта** возьмите:
- `svg/horizontal-light.svg` — в шапку
- `svg/favicon.svg` — как фавикон (плюс `png/favicon-32.png` для старых браузеров)
- Цвет `#3c3489` — для CTA-кнопок и акцентов

**Для App Store** возьмите:
- `png/icon-1024.png`

**Для Google Play** возьмите:
- `png/icon-512.png`

**Для PWA / Android** возьмите:
- `png/icon-192.png` и `png/icon-512.png`
- Подключите через `site.webmanifest` (пример в brand-guide.md)

## Что я бы делал дальше

1. **Сейчас:** регистрируйте `beatmetrics.ru` и `битметрика.рф`, ставьте фавикон на лендинг.
2. **На неделе:** проверьте Роспатент (ФИПС) на «BeatMetrics» / «БитМетрика», подавайте заявку на товарный знак.
3. **На месяце:** наймите дизайнера для доводки — он сделает анимированный splash, иллюстрации для лендинга, пиктограммы в едином стиле. Эти исходники для него хороший старт.
