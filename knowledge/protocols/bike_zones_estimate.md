# Велосипедные зоны — расчёт (2026-04-27)

Без отдельного лактатного / HRmax теста на велике — оценка из беговых зон через метод %HRR.

## Исходные данные

- **RHR** = 47 (по daily_stats Garmin)
- **Run HRmax** = 187 (наблюдаемый, по гонкам)
- **Run HRR** = 187 - 47 = **140**
- **Run zones** (из лактат-теста 25.04.2026):
  - Z1: <140 / Z2: 140-161 (АэП) / Z3: 162-178 (АнП) / Z4: 179-185 / Z5: 186+

## Эмпирические данные с велика (за год)

- 130 активностей, 82 ч
- Avg HR в среднем = 101 (очень лёгкие катания)
- Макс. avg HR = 140 (одна Z2-поездка)
- **Макс. наблюдаемый HR за всю историю = 153** (никогда не работал на пределе)

→ Реальный bike HRmax не известен. По типичному сдвигу для атлета с беговой доминантой: **bike HRmax ≈ run HRmax − 8 = 179** (диапазон 177-182).

## Метод: %HRR (Heart Rate Reserve)

Формула: `%HRR = (HR - RHR) / (HRmax - RHR)`

Принцип: переносим относительный уровень нагрузки между видами спорта через долю HRR, а не абсолютный HR.

### Шаги расчёта

1. Перевести каждую границу беговой зоны в %HRR (через Run HRR = 140)
2. Получить bike HR на той же %HRR через Bike HRR = 132 (= 179 - 47)

| Граница | Бег HR | %HRR (Бег) | Bike HR расч. |
|---|---|---|---|
| Z1/Z2 | 140 | (140-47)/140 = 66% | 47 + 0.66×132 = **134** |
| Z2/Z3 (АэП) | 161 | 81% | 47 + 0.81×132 = **154** |
| Z3/Z4 (АнП) | 178 | 94% | 47 + 0.94×132 = **171** |
| Z4/Z5 | 185 | 99% | 47 + 0.99×132 = **178** |
| HRmax | 187 | 100% | **179** |

## Итоговые вело-зоны (расчётные)

| Зона | HR | Назначение |
|---|---|---|
| Z1 | <134 | Recovery, разминка/заминка |
| Z2 | 134-154 | База, ССС, длительные катания (АэП ~154) |
| Z3 | 155-171 | Темповые на ПАНО (АнП ~171) |
| Z4 | 172-178 | Спец.выносливость, race-pace |
| Z5 | 179+ | Анаэробка, спринт |

|---|---|---|---|

Garmin занижает зоны на 10-20 bpm — потому что bike HRmax 172 это auto-detected из недонагруженных данных. LTHR 162 = 94% Garmin-Hmax — тоже не соответствует реальности.

## Применение в плане

**Май (вело-фокус):**
- Длительные вело Z1-Z2 → HR 120-150 (комфорт, ровное дыхание)
- Ускорения "в подъём" 30 сек → выйдет Z3-Z4 (160-180), нормально для коротких отрезков
- 2×10м под АэП 20.05 → целевой HR ~150-154

## Калибровка (рекомендуется)

### Простой тест: 5 минут максимально в подъём

1. Полная разминка 30+ мин Z1-Z2
2. 5 минут максимальное усилие в подъём 4-7° (ровный сильный темп, не до отказа на 1-й минуте)
3. Зафиксировать пиковый HR в последние 30 сек = реальный bike HRmax
4. Заминка 15+ мин

После теста перерасчёт через %HRR.

### Точнее: 20-минутный FTP тест

В подъёме / на ровной с постоянной мощностью. Даст:
- bike HRmax
- bike LTHR (= 95% от средней HR за 20 мин)
- FTP в Вт (если есть power-meter)

## Что менять в Garmin Connect

Settings → User Settings → Heart Rate Zones (Cycling) → Custom:

```
HRmax:  179 (с 172)
LTHR:   171 (с 162)

Z1: 100-133
Z2: 134-154
Z3: 155-171
Z4: 172-178
Z5: 179
```

После этого Training Effect и pulse-zone статистика на велике в Garmin будут корректнее.
