## Интерпретация — пошагово

### Шаг 1. HRmax

187 bpm — наблюдается на спринтерских финишах в Garmin.
Тест закончился на 181 (не до отказа), так что HRmax шире окна теста.

### Шаг 2. LT1 (аэробный порог, ~2 mmol/L) — линейная интерполяция

Соседние ступени:
- 13.5 км/ч → 1.9 ммоль / HR 158
- 14.0 км/ч → 2.7 ммоль / HR 162

Доля для лактата 2.0: `(2.0-1.9)/(2.7-1.9) = 0.125`
- скорость = 13.5 + 0.125×0.5 = **13.56 км/ч**
- HR = 158 + 0.125×4 = **159-160 bpm**

### Шаг 3. LT2 / ПАНО (~4 mmol/L)

Соседние ступени:
- 16.0 км/ч → 3.2 ммоль / HR 175
- 16.5 км/ч → 4.6 ммоль / HR 181

Доля для 4.0: `(4.0-3.2)/(4.6-3.2) = 0.571`
- скорость = 16.0 + 0.571×0.5 = **16.29 км/ч**
- HR = 175 + 0.571×6 = **178 bpm**

### Шаг 4. Z1/Z2 и Z4/Z5 — границы по лактатной таблице тренера

Тренер задаёт зоны границами лактата: <1.5 / 1.5-2.5 / 2.5-4.0 / 4.0-6.0 / >6.0.

**Z1/Z2 (лактат 1.5):**
- 10.5 → 1.3 / 131; 11.5 → 1.7 / 141
- доля = 0.5 → HR = **136**

**Z4/Z5 (лактат 6.0):** в тесте не дошли. Экстраполяция тренда (на верху +1.4 ммоль на 6 уд):
- HR(6.0) ≈ **187** — практически HRmax.
